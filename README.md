
# Crypto Price App

Shows Realtime Crypto Prices


## Features

- Sooth UI
- Realtime Price
- Fast Servers - No delay
- Incredible History Graph View


## Tech Stack

**Architecture Pattern:** MVVM

**Libraries Used:** Dagger-Hilt, Coroutines, Data binding, Graph View, Retrofit 

**Server:** CoinGecko
